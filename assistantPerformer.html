Assistant Performer

This Web MIDI application is normally tested in the latest stable versions of Firefox and Chrome on Windows 10.
Firefox requires the Jazz plug-in to be installed.
Note that it is not possible to display or play scores unless a valid MIDI output device has been selected, but that
only hardware or software devices that involve installing a plugin are currently supported.
On Windows, the free Virtual MIDI Synth can be installed and used in both browsers.
In Firefox, which depends on the Jazz plugin, the Microsoft GS Wavetable Synth (part of the Windows operating system) is available.
